ℹ️ このアプリについて

VulnWatchは脆弱性情報を収集・分析するためのPWA（Progressive Web App）です。

データソース:
・NVD (National Vulnerability Database) - 米国NISTが提供
・JVN (Japan Vulnerability Notes) - IPAとJPCERT/CCが提供

AI機能:
・Google Gemini APIを使用して脆弱性について質問・分析ができます

プライバシー:
・APIキーはサーバーに送信されません（ブラウザ内で直接APIと通信します）
・データはローカルのみに保持されます

📓 Notion連携

脆弱性トラッカーデータベースがNotionに用意されています。
NVD検索結果の各アイテムにある「📓」ボタンで、情報をコピーしてNotionに貼り付けできます。

使い方:
1. NVDで脆弱性を検索
2. 「📓」ボタンまたは「全件Notionにコピー」をクリック
3. Notionデータベースを開いて貼り付け

📱 PWAとしてインストール

このアプリはWebサーバーにホストすると、スマホやPCにインストールできるPWAになります。

🚀 Netlifyで公開（一番簡単）:
1. Netlify Drop を開く
2. vulnwatch-pwaフォルダをドラッグ＆ドロップ
3. 数秒で公開完了！URLが発行されます

🐙 GitHub Pagesで公開:
1. GitHubにリポジトリを作成
2. vulnwatch-pwaフォルダの中身をアップロード
3. Settings → Pages → Source: main → Save
4. 数分後にURLが発行されます

📲 スマホにインストール:
公開後、ブラウザで開いて「ホーム画面に追加」を選択
